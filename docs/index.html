Get Started in Minutes

Choose your preferred installation method

Install CLI Tool

# Install globally
npm install -g mcp-agentify

# Or use with npx
npx mcp-agentify --help

Quick Start

# Analyze your project
agentify analyze ./my-api

# Generate MCP server
agentify generate ./my-api --output ./mcp-server

# Interactive setup
agentify interactive

Install VS Code Extension

VS Code Marketplace

Search for "MCP Agentify" in VS Code Extensions

Install Extension

Command Line

code --install-extension saurabhkohli.mcp-agentify

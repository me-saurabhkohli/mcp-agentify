Installation

From VS Code Marketplace

  1. Open VS Code
  2. Go to Extensions view (Ctrl+Shift+X or Cmd+Shift+X)
  3. Search for "MCP Agentify"
  4. Click Install

From Command Line

code --install-extension saurabhkohli.mcp-agentify

Manual Installation

  1. Download the .vsix file from the releases page
  2. Open VS Code
  3. Run Extensions: Install from VSIX... from Command Palette
  4. Select the downloaded .vsix file

Usage

Command Palette

Open Command Palette (Ctrl+Shift+P or Cmd+Shift+P) and use these commands:

MCP Agentify: Generate MCP Server - Generate MCP server from current project
MCP Agentify: Analyze Project - Deep AI-powered project analysis
MCP Agentify: Interactive Setup - Launch guided setup wizard
MCP Agentify: Start MCP Debug Server - Launch debugging server
MCP Agentify: Show MCP Server Status - View real-time server health
MCP Agentify Debug: List MCP Tools - View available debugging tools
MCP Agentify Debug: Inspect MCP Context - Debug MCP context
MCP Agentify: Open Settings - Configure extension settings

Explorer Context Menu

Right-click on any folder in the Explorer to access generation options:

  • Generate MCP Server - Generate server from selected folder
  • Analyze with Agentify - Analyze folder structure
  • Configure Agentify - Create/edit configuration for this folder

Status Bar

The extension adds a status bar item showing:

  • Current project analysis status
  • Quick access to generation commands
  • Progress indicator during generation

Configuration

Extension Settings

Configure the extension through VS Code Settings:

{
  "mcp-agentify.preferredFormat": "typescript",
  "mcp-agentify.includeTests": true,
  "mcp-agentify.includeDocumentation": true,
  "mcp-agentify.enableSecurity": false,
  "mcp-agentify.enablePlugins": true,
  "mcp-agentify.autoAnalyze": true,
  "mcp-agentify.autoStartMCPServer": true,
  "mcp-agentify.mcpServerPort": 8890,
  "mcp-agentify.debugMode": false,
  "mcp-agentify.showNotifications": true
}

Workspace Configuration

Create an agentify.config.json file in your workspace root:

{
  "serverName": "my-api-server",
  "description": "Generated MCP server for My API",
  "version": "1.0.0",
  "outputFormat": "typescript",
  "includeTests": true,
  "includeDocumentation": true,
  "excludeEndpoints": [
    "GET:/health",
    "POST:/internal/*"
  ],
  "plugins": [
    "rest-analyzer",
    "template-engine"
  ]
}

Troubleshooting

Extension Not Working

  1. Check that Node.js is installed and accessible
  2. Verify the extension is enabled in Extensions view
  3. Reload VS Code window (Developer: Reload Window)
  4. Check Output panel for error messages

Generation Fails

  1. Ensure project structure is supported
  2. Check file permissions in output directory
  3. Verify configuration file syntax
  4. Enable verbose logging for detailed diagnostics

Performance Issues

  1. Exclude large directories in .gitignore
  2. Use specific project paths instead of workspace root
  3. Disable unnecessary plugins in configuration
  4. Consider using CLI for very large projects
